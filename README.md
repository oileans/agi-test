# agi-test


# Setup:

  1. Instalar o bundler pelo terminal
  ```
  gem install bundler
  ```
  2. Instalar as dependencias do projeto
  ```
  bundle install
  ```
  3. stable version chromedriver
  https://googlechromelabs.github.io/chrome-for-testing/#stable

# Rodar feature:

  ```
  cucumber -t @agibank  
  ```
